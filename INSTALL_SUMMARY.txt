💻 SETUP EN 5 MINUTES

1. Obtenir clé API Claude
   👉 https://console.anthropic.com/

2. Fork le repo sur GitHub
   👉 Tu recevras un lien

3. Ajouter secret ANTHROPIC_API_KEY
   👉 Settings → Secrets → New secret

4. Activer le workflow
   👉 Actions → Enable

5. Test (optionnel)
   👉 Run workflow manuellement

📝 Voir QUICKSTART.md pour détails
